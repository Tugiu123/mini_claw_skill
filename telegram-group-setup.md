# Telegram 群组配置指南

让 OpenClaw/Telegram 机器人在群组中正常响应。

## 问题

机器人被拉进 Telegram 群组后不回复消息。

## 解决方案

### 1. 关闭 Private Mode

在 @BotFather 中：

1. 发送 `/setprivacy`
2. 选择你的机器人
3. 选择 **Disable**（禁用）

> 关闭后，机器人可以读取群组中所有消息（不仅是 @它 的）

### 2. 群组授权

1. 打开群组 → 点击群名称 → **管理**
2. **管理员** → **添加管理员**
3. 选择机器人
4. 确保开启 **发送消息** 权限

### 3. 验证

在群里 @机器人 或发送消息测试。

## 配置命令参考

| 命令 | 作用 |
|------|------|
| `/setprivacy` | 设置隐私模式 |
| `/setjoingroups` | 设置是否允许被拉入群组 |
| `/setname` | 设置机器人名称 |
| `/setdescription` | 设置机器人描述 |

## 注意事项

- 机器人默认是 **隐私模式**，无法读取群组消息
- 必须在 BotFather 关闭隐私才能在群组中正常工作
- 群组中需要至少有一个管理员权限才能正确配置
